## Source Code Overview

This directory contains core implementations for:

- DG modal/nodal transformations
- SIAC kernel construction and convolution
- Basis and quadrature utilities

### Attribution

Parts of the nodal-to-modal transformation and SIAC routines are inspired by:

Jennifer K. Ryan's SIAC MATLAB test codes:
https://github.com/jennkryan/SIACMagicTestCodes/

The implementations here are rewritten and extended for Python, tensor-product DG methods, and 2D data processing.
